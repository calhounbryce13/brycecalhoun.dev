Research: Unlocking Health Insights with Computer-Vision

Computer vision is a new and exciting field within the context of artificial intelligence that comes with lots of opportunities for valuable applications. It's only natural to wonder if computer vision could help us understand our own health better. This was the exact purpose for this longitudinal study. The data preprocessing included publicly available Twitter data that took well over a year's time to finilize. Some key takeaways from this project are that while unsurprisingly, the human eye remains the leading authority in the detection of visual health outcomes and health-contributing environmental features, there is a massive potential in the refinement of computer vision models for the identification of complex emotional health outcomes. This is especially the case in the context of children in schools. For a more detailed summary here is a link to the full presentation slide.

Skills : libraries

React is a popular JS library for building user interfaces (UI), particularly for single-page applications (SPAs). React follows a declarative approach, where developers describe how the UI should look based on the state, and React takes care of updating the view when the state changes. React's component-based architecture also promotes maintainability and scalability for complex applications.

Tensorflow is a great resource for pre-built ML models and AI algorithms. The combination of Tensorflow and Keras allows developers to built custom neural networks in python code and train them on either publicly available datasets or your own built data sets.

OpenCV is another widely used and highly regarded library in the field of AI, specifically for computer vision applications. It offers a comprehensive set of tools that enable developers to process and manipulate image data, which is essential for building and training computer vision models.

Matlabplotlib in the context of machine learning, the ability to visualize data is critical for understanding patterns and insights. This library is incredibly useful for displaying various types of graphs and charts, such as line plots, bar charts, histograms, and scatter plots, based on complex datasets. By utilizing these visualization methods, its easier to interpret the results of machine learning models, identify trends, and communicate findings effectively. Whether you're exploring data, tuning models, or presenting results to stakeholders.

NumPy is a powerful Python library used for numerical computing, particularly for handling large, multi-dimensional arrays and matrices. It provides a wide range of mathematical functions to operate on these arrays efficiently. The library supports element-wise operations, linear algebra, random number generation, and more.

AI Minimax algorithm for Tic-Tac-Toe game logic

Starting with the basics of artificial intelligence algorithms, adverserial serach is a key component of implementing the human decision-making process. adverserial search deals with navigating to one of various optimal goal states ( a win for the agent ) and moving the agent closer and closer with each turn. Moves that place the agent closer to the goal state will be rewarded and moves that deter the agent from reaching the goal state are not. The agent will then attempt to either maximize or minimize a score based on the moves that it makes.
